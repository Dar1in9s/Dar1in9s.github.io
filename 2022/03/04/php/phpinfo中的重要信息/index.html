phpinfo中的重要信息
2022-03-04|php
字数总计: 237|阅读时长: 1 分钟

phpinfo页面中含有大量有用的信息，这对往后的测试有很大帮助。

基础信息

版本信息、系统类别和系统版本、编译信息。

php配置文件位置：Loaded Configuration File

php支持的流：Registered PHP Streams

支持的流过滤器：Registered Stream Filters

核心信息Core

是否允许远程包含：allow_url_fopen和allow_url_include

禁用的类和函数：disable_classes和disable_functions

错误提示：display_errors、error_reporting

include_path：文件包含时的默认路径

open_basedir：目录限制

short_open_tag：是否支持短标签

upload_tmp_dir：上传文件临时存放路径

session

session.name：session名字

session.save_path：session保存的路径

session.serialize_handler：定义用来序列化/反序列化的处理器名字

session.upload_progress.enabled：上传大文件时开启session，便于返回进度信息

Environment

这里可以看到系统的环境变量

PHP Variables

$_SERVER[‘SCRIPT_FILENAME’]：phpinfo的绝对路径

$_SERVER[‘REMOTE_ADDR’]：显示该网站的真实的 ip 地址

$_SERVER[“DOCUMENT_ROOT”]：网站根目录
